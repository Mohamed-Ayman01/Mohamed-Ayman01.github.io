# Mohamed E. Ayman — Front-End Developer Portfolio

This is a professional Front-End Developer portfolio built using React, Vite, Framer Motion, and Tailwind CSS.

## Tech Stack
- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (v3)
- **Animations:** Framer Motion
- **Hosting:** GitHub Pages via GitHub Actions
